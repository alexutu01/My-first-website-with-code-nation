3.4 Describe how you would go about testing digital solutions

Begin by making sure that every button on every screen works. You also need to ensure that you can enter simple text into each field without crashing the software, this metod is known as alpha testing. beta testing is another way that your code can be tested, having another pair of eyes looking at the source code can uncover a lot of problems that you might have missed. Make time to run as many of these tests as possible before you hand the code over to anyone else. It is always important to test in your terminal if you are writing python or any other similar language. And if it is HTML-CSS, refresh the page as many times neede or after every change made to the code, to see the final result.
